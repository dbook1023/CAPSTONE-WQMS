Admin Task FAQs

How do I move or swap a physical sensor device to a different fountain?

To transfer a physical ESP32 device, simply go to the Hardware Registry page, click Register Device (or edit the device), type in the existing Serial number (MAC address), select the new fountain, and click save. The system automatically unbinds it from the old fountain and assigns it to the new one. All future water readings from this device will instantly route to the new fountain, while historical logs remain safely archived.

How do I calibrate the sensors remotely?

Remote calibration is handled via the "Fountain Status" admin panel. Select the specific node and input the calibration offset values. The ESP32 will update its local logic on the next heartbeat.

Can I restrict user access to certain reports?

Yes. Access control is managed in the "User Management" section. You can toggle "Report Access" for individual employees based on their department requirements.

What happens if a sensor node goes offline?

If the system detects no heartbeat for more than 15 minutes, an automatic "System Offline" alert is triggered and sent to all administrators via the dashboard.
