Frequently Asked Questions

How often is the water quality data updated?

Data is transmitted in real-time from our ESP32-based sensor nodes every 5 minutes. The dashboard updates automatically to reflect the latest readings.

What do the different alert levels mean?

Ideal: Parameters are within the safe OLFU standard range.
Warning: One or more parameters are approaching the limit.
Critical: Immediate action is required as parameters are unsafe.

How can I export the monitoring reports?

Navigate to the "Reports" page. You can filter by date range and use the "Premium Download" button to save reports as PDF or CSV files.
