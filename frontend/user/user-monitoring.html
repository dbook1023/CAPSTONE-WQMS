Live Monitoring Guide

Step-by-step instructions on how to operate the Water Quality Monitoring System (WQMS).

1

Select Drinking Fountain

Click "Start Reading" in the top header and pick a fountain from the dropdown or quick-lookup by ID (e.g., F001).

2

Monitor Real-time Telemetry

Watch real-time sensor streams (pH, Turbidity, Temperature, TDS) update on highly stylized, color-coded dashboard cards and line graphs.

3

Compare Saved Readings

Click "Save Snapshot" at any point to save a telemetry slice. These slices will appear side-by-side in the Session Readings Comparison table at the bottom of the screen.

4

Export Monitoring Report

Click "Generate Report" to aggregate all captured data points during the active session into a printable PDF report for submission or archive.
